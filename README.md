## Develop 🔧
- To run dev server: ```gatsby develop```
  - Hot reloading dev environment available from ```localhost:8000```

## Build & Deploy :zap:
- Build with:  ```gatsby build```
- Add favicons to public folder
- Deploy public folder to webserver of choice 🆒
